**文档版本**：v0.3  
**创建日期**：2026-04-24  
**当前规则系统**：克苏鲁的呼唤（Call of Cthulhu），参考《阿卡姆恐怖》剧情风格  

---

## 一、产品概述

一款供 TRPG 游戏主持人（KP）在线下桌游时使用的网页端氛围音乐播放器。主持人通过简单操作切换场景 BGM、触发即时音效，为玩家创造沉浸式的克苏鲁恐怖氛围。

---

## 二、目标用户

| 角色 | 描述 |
|------|------|
| 主要用户 | 游戏主持人（KP），线下使用，单设备操作 |
| 受益用户 | 同桌玩家，被动聆听，无需操作设备 |

**使用环境**：线下面对面，主持人笔记本/平板开启网页，声音通过音响或耳机输出。

---

## 三、核心功能

### 3.1 场景 BGM 播放
- 提供多个预设场景，**每个场景对应 3 首候选曲目**，点击场景时从中随机抽取一首循环播放
- 同一场景连续切换时，尽量避免随机到上一首相同曲目（排除上次播放的曲目后再随机）
- 场景切换时，旧 BGM 淡出，新 BGM 淡入（过渡时长约 2 秒）
- 支持手动调节 BGM 主音量

### 3.2 即时音效触发（SFX）
- 界面上排列若干音效按钮，点击即播放
- 音效为一次性播放，不循环
- 支持手动调节音效主音量

### 3.3 自动音量闪避（Audio Ducking）⭐ 核心交互
- 触发任意音效时，BGM 音量自动在 0.3 秒内压低至原音量的 30%
- 音效播放结束后，BGM 在 1.5 秒内淡回原音量
- 此行为自动发生，无需主持人手动操作

### 3.4 紧急静音
- 提供一键"全部静音"按钮，快速切断所有声音（应对突发情况）

## 六、界面设计要求

### 整体风格
- **主色调**：深色系（黑色/深灰/暗棕），配合 1920 年代纸张黄作为点缀色
- **字体风格**：古典衬线体，营造旧报纸/古籍感
- **无多余装饰**，操作优先，保证主持人能在黑暗环境中快速点击

### 布局结构
```
┌─────────────────────────────────────────┐
│  [Logo/标题]              [全部静音] 🔇  │
├─────────────────────────────────────────┤
│  [ 🎵 场景选择 ▾ ]  当前：[场景名]       │
│  ↓ 点击展开下拉列表（overlay 浮层）       │
│  ┌─────────────────────────────────┐    │
│  │ 01 阿卡姆街道                    │    │
│  │ 02 米斯卡托尼克大学  ← 当前高亮  │    │
│  │ 03 调查现场                      │    │
│  │ ...（共10个，可滚动）             │    │
│  └─────────────────────────────────┘    │
│  BGM 音量：████████░░                   │
├─────────────────────────────────────────┤
│  即时音效区                              │
│  [环境] [室内] [人物] [克苏鲁]  分组标签 │
│  [雷声] [大雨] [风声] ...  音效按钮网格  │
│                           ████████░░ 音量│
└─────────────────────────────────────────┘
```

### 交互要求
- 场景切换：点击"场景选择"按钮展开下拉浮层，选中场景后浮层自动关闭，当前激活场景在浮层内高亮显示
- 下拉浮层：以 overlay 形式覆盖在主界面上方，点击浮层外区域可关闭
- 音效触发：单击音效按钮即播放，支持快速连点（每次点击独立触发一个新的音频实例，多个同名音效可同时叠加播放），播放中按钮有视觉反馈（高亮/动画）
- 音量调节：滑块控制，BGM 和 SFX 分别独立调节
- 页面不需要登录/账号，直接打开即用

## 七、技术要点备注

| 技术需求 | 实现思路 |
|---------|---------|
| BGM 循环播放 | HTML5 Audio API，loop 属性 |
| 场景切换淡入淡出 | Web Audio API 的 GainNode，渐变音量 |
| Audio Ducking | 检测 SFX 播放事件，动态调整 BGM GainNode |
| 音源 | Incompetech（BGM）+ Freesound（SFX），预先选定 URL/文件 |
| 部署 | 静态网页，Vercel 免费部署，无需后端 |
